pxLevels adds a new leveling system for your users,
it also allows you to add rewards to levelups.

## Commands

`<>` = Required `[]` = Optional

Command                              | Permission                      | Description
------------------------------------ | ------------------------------- | -----------
`/levels`                            | None                            | Opens the pxLevels GUI
`/adminexp help`                     | core.command.adminexp           | Brings up the pxLevels XP Admin commands
`/adminexp add`                      | core.command.adminexp           | Allows you to add XP to specified user
`/adminexp set`                      | core.command.adminexp           | Allows you to set XP for a specified user
`/adminexp remove`                   | core.command.adminexp           | Allows you to remove XP from a specified user
`/alevels help`                      | core.command.adminlevel         | Brings up the help guide for Levels in pxLevels
`/alevels listlevels`                | core.command.adminlevel         | Allows you to list the available levels
`/alevels reload`                    | core.command.adminlevel         | Allows you to reload pxLevels
